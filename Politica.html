Política de Privacidad

Última actualización: abril 2026

1. Qué datos recoge esta extensión

La extensión accede a los siguientes datos durante su uso:

  • El número de teléfono del contacto activo en WhatsApp Web.
  • Los últimos mensajes de la conversación activa.
  • La actividad de navegación dentro de WhatsApp Web para detectar cambios de conversación.

2. Cómo se usan los datos

Los datos se envían exclusivamente al sistema interno de la empresa para consultar información comercial del cliente y generar recomendaciones para el equipo de ventas. No se utilizan para ningún otro fin.

3. Almacenamiento

La extensión no almacena datos de conversaciones ni números de teléfono de forma local ni en servidores propios de la extensión. Los datos se transmiten en tiempo real al sistema interno y no se retienen.

4. Compartir datos con terceros

Los datos no se venden, ceden ni transfieren a terceros fuera del sistema interno de la empresa.

5. Uso interno

Esta extensión es de uso exclusivo para el equipo de ventas de la empresa. No está disponible públicamente para usuarios externos.

6. Contacto

Para consultas sobre esta política, contactarse con el administrador del sistema.
